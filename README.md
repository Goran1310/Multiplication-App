# 🎮 Multiplication Master

An interactive, kid-friendly multiplication practice app designed for children aged 8-10.

## ✨ Features

✅ **Score Counter** - Track your progress as you answer correctly  
✅ **Star Rewards** - Get a ⭐ every 5 correct answers  
✅ **Sound Effects** - Built-in sounds for correct and wrong answers (no files needed!)  
✅ **Timer Mode** - 30-second challenge for extra fun  
✅ **Smart Hints** - Shows repeated addition when you get it wrong  
✅ **Colorful Design** - Big, bright, easy-to-read interface  
✅ **Choose Your Table** - Pick any times table from 1-10  

## 🚀 How to Start

1. **Open the app**
   - Simply double-click `index.html` in your file explorer
   - Or open it in any web browser (Chrome, Firefox, Edge, Safari)

2. **No installation needed!**
   - Works completely offline
   - No internet connection required
   - No downloads or setup

## 🎯 How to Play

1. **Pick a times table** by clicking one of the number buttons (1-10)
2. **Type your answer** in the big input box
3. **Press Enter** or click "Check Answer"
4. **Get instant feedback** with sounds and emojis:
   - ✅ Correct = Score +1 and new question
   - ❌ Wrong = Helpful hint appears
5. **Collect stars** ⭐ - Get one every 5 correct answers!
6. **Try Timer Mode** for a 30-second challenge

## 🔊 About Sounds

The app uses **Web Audio API** to generate sounds:
- ✅ **Correct answer**: Pleasant two-note "ding"
- ❌ **Wrong answer**: Gentle "buzz"
- 🎵 **No files needed** - sounds are generated by the browser!

> **Note**: If sounds don't play, click anywhere on the page first. Some browsers require user interaction before playing audio.

## 🎨 Features Explained

### Score System
- Start at 0
- Each correct answer adds 1 point
- Score persists until you click "New Game"

### Star Rewards
- Earn a ⭐ at 5, 10, 15, 20, 25... points
- Stars appear next to your score
- Celebrate animation when you earn a star!

### Hints
When you get an answer wrong, the app shows:
```
💡 Hint: 5 × 3 means 3 groups of 5
That's: 5 + 5 + 5 = 15
```

### Timer Mode
- Click "⏱️ Timer Mode" to start
- 30-second countdown
- Answer as many as you can!
- Timer turns red at 10 seconds
- Final score shown at the end

## 🎨 Customization Ideas

Want to modify the app? Here are some easy changes:

### Change Timer Duration
In `app.js`, find this line:
```javascript
timeLeft = 30;
```
Change `30` to any number of seconds you want.

### Change Star Frequency
In `app.js`, find this line:
```javascript
if (score > 0 && score % 5 === 0) {
```
Change `5` to earn stars more or less often.

### Add More Times Tables
In `index.html`, add more buttons:
```html
<button class="num-btn" onclick="selectNumber(11)">11</button>
<button class="num-btn" onclick="selectNumber(12)">12</button>
```

### Change Colors
In `style.css`, modify the gradient backgrounds:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
Use [uiGradients](https://uigradients.com/) to find beautiful color combinations.

## 🐛 Troubleshooting

**Sounds not working?**
- Click anywhere on the page first
- Check browser volume settings
- Try a different browser

**Layout looks broken?**
- Make sure all three files are in the same folder
- Clear browser cache (Ctrl+F5)
- Try a different browser

**Timer stuck?**
- Click "New Game" to reset
- Refresh the page (F5)

## � Deployment

### Netlify Deployment

This app is deployed on Netlify. Here's what you need to know:

**Important Lesson Learned - Vite Permission Error Fix**

If you encounter `sh: 1: vite: Permission denied` errors during Netlify builds, the solution is to execute Vite directly with Node.js instead of using the binary wrapper:

**In `package.json`:**
```json
"scripts": {
  "build": "node node_modules/vite/bin/vite.js build"
}
```

**Why this works:**
- The `vite` command is a shell wrapper that may lack execute permissions on Linux-based CI/CD systems
- `node` directly executes the JavaScript file, bypassing permission requirements
- Node.js always has proper permissions in Netlify's build environment

**Alternative approaches that DON'T work:**
- ❌ `vite build` - Permission denied error
- ❌ `npx vite build` - Still triggers permission error
- ❌ Changing file permissions - Not possible in Netlify environment

**Netlify Configuration (`netlify.toml`):**
```toml
[build]
  command = "npm run build"
  publish = "dist"

[build.environment]
  NODE_VERSION = "20"
```

**Deployment Steps:**
1. Connect your GitHub repository to Netlify
2. Netlify auto-detects the build settings from `netlify.toml`
3. Push changes to trigger automatic deployment
4. Build completes successfully with the direct Node.js execution method

This solution is applicable to any Vite project deployed on Netlify or similar CI/CD platforms that may have restrictive file permissions.
